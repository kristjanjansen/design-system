# design-system

Agent-first React design system. oklch colors, CSS custom properties, multi-brand theming, native web platform features.

## For consumers

```
Install design-system and create a settings page with name, email (required), bio fields, a "receive notifications" switch, and save/cancel buttons. Read node_modules/design-system/CLAUDE.md and DESIGN.md for conventions.
```

## For library contributors

```
Add a Radio component. Follow the conventions in CLAUDE.md and DESIGN.md. Consult the Checkbox component for patterns.
```
